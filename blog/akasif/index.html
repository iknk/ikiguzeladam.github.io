How to add lwjgl libraries in to a Java Project

Tools to download;
 • LWJGL 2

How to setup the library;
First you should know that this process has to be done everytime you create a new project, until you assign it default. Setting it default would be less beneficial, because when we dont need it, we still had to use it. So, lets start:
 •  Setup a project
 •  Create a lib folder and inside lib folder create 2 folders named jar, natives.
 •  Extract the jar files you had downloaded to the lib/jar folder.
 •  Extract the native files you had downloaded to the lib/natives folder with related os.
 •  Now open your project, go properties, find Java build path, and add external jar.Choose the jar files that you previously located in lib/jar.
 •  After you are done about it, go click lwjgl.jar and choose add natives. Search for your natives that you previously located in lib/natives and add them as natives.
 • You are ready to go!
